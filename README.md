Рекомендуется использовать PyQt6.
На момент защиты для корректной работы было принято решение заменить PyQt6 на PyQt5 так как 6 версия не скачивается на
рабочий ноутбук (Который находится в IT-кубе).

## Структура

1) X256 - алгоритм шифрования с тестами.
2) ui - папка в .ui файлами, созданными с помощью QtDesigner.
3) SixteenNumbersGenerator и key - файлы, для генерации ключей, необходимых для шифрования.
4) interfaces - там находятся все файлы, которые отвечают за PyQt.
5) main.py файл, необходим только для удобного запуска.
6) requirements.txt - файл с библиотеками

## Запуск со сборкой

0) (Создайте или войдите в venv и определите интерпритатор)
1) pip install -r requirements.txt
2) python main.py
